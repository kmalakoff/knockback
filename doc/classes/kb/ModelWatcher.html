    Class: kb.ModelWatcher

    Defined in: src/knockback-core/knockback-model-watcher.coffee

    Overview

    Used to provide a central place to aggregate registered Backbone.Model events rather than having all kb.Observables register for updates independently.

    Class Method Details

    + (void) useOptionsOrCreate(options, model, obj, callback_options)

    Used to either register yourself with the existing model watcher or to create a new one.

    Parameters:

    • options (Object) — please pass the options from your constructor to the register method. For example, constructor(model, options)
    • obj (Backbone.Model|Backbone.ModelRef) — the Model that will own or register with the store
    • owner (ko.observable|Object) — the owners of the model watcher
    • callback_options (Object) — information about the event and callback to register

    Options Hash: (options):

    • model (Function) — callback for when the model changes (eg. is loaded). Signature: function(new_model)
    • update (Function) — callback for when the registered event is triggered. Signature: function(new_value)
    • event_name (String) — the name of the event.
    • key (String) — the optional key to filter update attribute events.
